# Bili Favorites Classifier

一个用于读取 Bilibili 用户公开收藏夹、抓取视频标签并进行分类展示的桌面工具。

当前版本：`v1.3.0`

## 功能

- 输入 Bilibili 用户 ID，读取其公开收藏夹中的视频 BV 号列表
- 自动抓取视频标签与视频分区
- 默认模式下按照 Bilibili 视频分区直接分类
- 自定义模式下支持不限数量的类别，每个类别可填写多个关键 tag
- 同一视频可同时命中多个自定义类别，未命中的视频统一进入“未分类”
- 图形界面展示分类结果，双击视频即可在浏览器打开对应页面
- 使用更现代化的桌面界面布局，分类、账号同步和结果浏览分区更清晰
- 支持右键将视频移动到其他分类
- 支持粘贴登录 Cookie，检测当前登录账号
- 支持将分类结果同步到 B 站收藏夹，可选择“复制”或“移动”
- 支持将分类结果导出为 `JSON` 或 `CSV`
- 支持通过 `PyInstaller` 打包为 Windows `exe`

## 运行方式

运行`run.bat`

## 打包为 exe

项目根目录已经提供：

- `build_exe.bat`
- `BiliFavoritesClassifier.spec`

在 Windows 下直接双击或在终端运行：

```bat
build_exe.bat
```

构建完成后，生成文件位于：

```text
dist\BiliFavoritesClassifier.exe
```

说明：

- 打包脚本会自动创建独立的构建虚拟环境 `.venv-build`，不会依赖你当前全局 Python 的打包环境。
- 若首次构建较慢，通常是正常现象，因为需要下载安装打包依赖。

## 使用说明

1. 在窗口顶部输入 Bilibili 用户 ID。
2. 选择“默认分类”或“自定义分类”。
3. 如果使用自定义分类，可通过“新增分类”持续添加类别，并填写类别名称与关键 tag。
4. 点击“抓取并分类”。
5. 结果会在右侧树形表格中展示，双击某条视频即可打开 Bilibili 页面。
6. 右键某条视频可将其移动到其他分类。
7. 点击“保存结果”可导出本地文件。
8. 若要把分类结果同步到 B 站，在左侧“账号同步到 B 站”区域粘贴完整 Cookie。
9. 点击“检测登录”确认当前账号。
10. 选择“复制到分类收藏夹”或“移动到分类收藏夹”。
11. 选择新建收藏夹的公开/私密属性，然后点击“应用分类结果到 B 站”。

## 自定义分类规则

- 每一行代表一个类别。
- 左侧填写类别名，右侧填写关键 tag，关键词用英文逗号分隔。
- 规则主要基于视频 tag 做匹配，支持相同 tag 与相近 tag 的匹配。
- 同一个视频可以同时归入多个类别。
- 未命中任何类别的视频会统一进入“未分类”。

## 同步到 B 站收藏夹

- `复制到分类收藏夹`：更安全，会把视频复制到新分类收藏夹中，原收藏夹内容不动。
- `移动到分类收藏夹`：会把视频从原收藏夹迁移到分类收藏夹中。
- 若某个视频在自定义模式下同时命中多个分类，程序会禁止执行“移动”，并提示你改用“复制”或先手动调整分类。
- 当前版本仅支持把结果同步回“与已分类 UID 相同”的登录账号，避免误操作到其他账号。

## 如何获取 Cookie

1. 打开浏览器并登录目标 B 站账号。
2. 访问 `https://www.bilibili.com`。
3. 按 `F12` 打开开发者工具，切到 `Network` 面板并刷新页面。
4. 点开任意一个发往 `bilibili.com` 的请求，在 `Request Headers` 中找到 `Cookie`。
5. 复制整段 `Cookie` 字符串，粘贴到本工具。

最低需要的关键字段：

- `SESSDATA`
- `bili_jct`

安全提醒：

- Cookie 等同于登录态，不要发给别人。
- 建议只在本地临时粘贴使用，用完后清空。

## 说明

- 本项目使用 Bilibili 公共接口获取公开收藏夹和标签数据；如果目标用户没有公开收藏夹，工具会直接提示。
- 默认分区分类依赖 Bilibili 返回的 `tid/tid_v2` 信息；个别视频若接口未返回完整分区名，会回退到内置分区映射或分区编号显示。
- 收藏夹写操作依赖登录 Cookie 与 `bili_jct`，也依赖 Bilibili 当前接口规则；若 B 站后续调整接口，可能需要同步更新。
- 结果依赖公开数据和接口可访问性，若 Bilibili 后续调整接口，可能需要同步更新。
